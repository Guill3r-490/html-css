Anatomia do tipo

Certos tipos de fontes podem trazer mais familiariedade com certas palavras do que outras. Para analisar a anatomia desses tipos é essêncial, que se verifique o tamanho de cada letra na fonte utilizada.

Letra "X"

Ela é o ponto de partida para ciração de qualquer fonte, pois a sua altura define a altura base de todas as letras minúsculas.
Assim como existe uma altura padrão para as letras minúsculas, existe para as maiúsculas.

Altura ascendente e descendete

São as alturas que sobram além da altura base de seus respectivos padrões (Ex.: k,b,g, Q).
Descendente= altura que sobra para baixo
Ascendente= altura que sobra para cima

Corpo

Soma de todas as alturas

Anatômicos Corporais

Parte descendete da fonte

Olho

Grande espaço internos e fechado por todos os lado

Barriga

Junção de dois arcos

Espinhas

Curva que apenas existe na letra "s"

Pé

Base da fonte (onde fica as serifas)

Perna

Elemento que sai de uma haste ou braço e vai para baixo

Braço

Elemento que sai de uma haste e vai para cima

Anatômicos Geometricos

Terminal

Qunado se tem um componente que nâo está ligado a outro lugar

Vértice

São ângulos das letras

Arco

São cruvas, presente (por exemplo) na letra n.

Esporão

São "saltinhos de sapato" que servem para dar um impressão de base para a letra

Hastes

São grandes partes retas das letras

Filetes

Une duas hastes ou arcos

Serifas

São pequenos traços gerados em apenas em algumas fontes. Tem como função de criar uma linha imaginária de forma irracional para ajudar na leitura, pois lemos as palavras como um todo e não letra por letra.
